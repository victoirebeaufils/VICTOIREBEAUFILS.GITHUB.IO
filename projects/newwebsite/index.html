A new website

See Github repo or view demo

This website served its purpose for about a year, but I think it's time for a redesign. Here are some features I'd like to implement.

Features

  • Sleek and minimalistic theme

    I want to work on creating a theme that allows the user to focus more on the content rather than on the features of a website alone. I also want to improve time performance when rendering.

  • Programming projects focused

    Last year I struggled with figuring out where to showcase both my artwork and my programming projects. This year, I've decided to sort out the artwork component of my main website and to relocate it to another website dedicated for my artistic endeavours.

  • Responsiveness

    Despite the numerous advantages of Bootstrap, one thing that the framework does not do automatically is making the website responsive, apart from resizing elements. Responsiveness was not one thing I focused on last year, but as technology is evolving and the public is shifting towards mobile devices to access the internet, I thought it should be something to focus on.

  • Accessibility

    This was already a topic I was interested in last year, and my interest has only grown as I've worked towards familiarizing myself with different disabilities and the various ways technology could account for them and provide a better user experience. I'd like to try making a website that can be navigated by all without any design flaws that could impede one's ability to view its content.

Goals

  • Finishing the theme and creating a style guide
  • Transforming my CSS code into SASS code
  • Using React and React Bootstrap to minimize repeated code
  • Create a backend to store project information and use Gatsby to pull data from the created database
  • Create a page/form to publish project articles more easily
